支持的 HTTP 方法

GET 获取资源

请求获取指定资源的内容。这是最常用的HTTP方法。

// 示例请求 curl http://localhost:7878/index.html // 响应 HTTP/1.1 200 OK Content-Type: text/html Content-Length: 1234 Date: Sat, 11 Jan 2026 12:00:00 GMT Server: eslzzyl-webserver <html>...</html>

HEAD 获取响应头

与 GET 相同，但只返回响应头，不返回响应体。用于检查资源是否存在或获取元数据。

// 示例请求 curl --head http://localhost:7878/image.jpg // 响应（仅头部，无body） HTTP/1.1 200 OK Content-Type: image/jpeg Content-Length: 335872 Date: Sat, 11 Jan 2026 12:00:00 GMT Server: eslzzyl-webserver

OPTIONS 查询支持的方法

查询服务器支持哪些 HTTP 方法。

// 示例请求 curl -X OPTIONS http://localhost:7878 -i // 响应 HTTP/1.1 204 No Content Allow: GET, HEAD, OPTIONS Date: Sat, 11 Jan 2026 12:00:00 GMT Server: eslzzyl-webserver
Warning: 当前版本不支持 CORS 预检请求 (preflight requests)。

POST 提交数据

向服务器提交数据（部分支持，主要用于 PHP 处理）。

HTTP 状态码

服务器返回的常见状态码及其含义：

200 OK 请求成功，返回请求的资源内容
204 No Content 请求成功，但没有返回内容（用于 OPTIONS 请求）
400 Bad Request 请求格式错误或无法解析
404 Not Found 请求的资源不存在
405 Method Not Allowed 不支持的 HTTP 方法
500 Internal Server Error 服务器内部错误（如 PHP 执行失败）
501 Not Implemented 请求的 HTTP 版本不支持

响应头说明

响应头 说明 示例值
Content-Type 资源的 MIME 类型 text/html, image/jpeg, application/json
Content-Length 响应体的字节长度 1234
Content-Encoding 内容编码方式（压缩） br, gzip, deflate
Date 响应发送时间（UTC） Sat, 11 Jan 2026 12:00:00 GMT
Server 服务器标识 eslzzyl-webserver
Allow 允许的 HTTP 方法 GET, HEAD, OPTIONS

压缩支持

自动内容协商

服务器根据客户端的 Accept-Encoding 请求头自动选择最优压缩算法。

  • Brotli (br) - 最高压缩率，适合现代浏览器
  • Gzip (gzip) - 广泛支持，压缩效果好
  • Deflate (deflate) - 兼容性最好
// 请求示例（浏览器自动发送） GET /index.html HTTP/1.1 Host: localhost:7878 Accept-Encoding: br, gzip, deflate // 响应（使用 Brotli 压缩） HTTP/1.1 200 OK Content-Type: text/html Content-Encoding: br Content-Length: 456 // 压缩后大小

特殊功能

Directory Browsing

当请求路径指向目录时，服务器返回该目录的文件列表。

  • 自动排序（文件夹优先，名称排序）
  • 显示文件大小和修改时间
  • 支持超链接导航
  • 优雅的表格展示

LRU Cache

服务器维护一个 LRU（最近最少使用）文件缓存，自动缓存常访问的文件。

  • 自动缓存热门文件
  • 检测文件修改时间
  • 缓存大小可配置
  • 提升响应速度 300%+

PHP Support

支持执行 .php 文件（需要系统安装 PHP）。

// 访问 PHP 文件 curl http://localhost:7878/php/time.php // 服务器调用 php-cgi 执行并返回结果
Security Warning: PHP 支持仅用于课程演示，没有安全防护，请勿在生产环境使用！

配置说明

服务器配置文件位于 config/development.toml：

www_root = "./static/" # Web 根目录 port = 7878 # 监听端口 worker_threads = 0 # 工作线程数（0=自动） cache_size = 10 # 缓存容量（文件数） local = true # 是否本地模式（127.0.0.1）
